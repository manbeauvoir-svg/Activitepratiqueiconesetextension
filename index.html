Activité 1 / 4
Quiz rapide
6 questions pour vérifier vos connaissances sur les icônes et les extensions.
Q1
Que signifie la petite flèche dans le coin inférieur gauche d'une icône ?
Réponse : raccourci. La petite flèche est l'indicateur visuel universel d'un raccourci. Supprimer le raccourci ne supprime pas le fichier original.
Q2
Quelle extension est associée à un fichier Excel ?
Réponse : .xlsx — .docx = Word, .pptx = PowerPoint, .pdf = document universel.
Q3
À quoi reconnaît-on un dossier compressé dans l'Explorateur ?
Réponse : fermeture éclair. C'est le symbole visuel des dossiers compressés (.zip). La flèche, elle, signale un raccourci.
Q4
Vous recevez un fichier nommé "note.pdf.exe". Que faut-il faire ?
Réponse : ne pas l'ouvrir. Seule la dernière extension compte (.exe = programme). C'est une technique courante pour piéger les utilisateurs.
Q5
Quel est le principal avantage du format PDF pour partager un document ?
Réponse : préserve la mise en page. C'est le format universel idéal pour partager un document dont on ne veut pas modifier l'apparence.
Q6
Quelle extension correspond à un fichier image ?
Réponse : .jpg — .mp3 = audio, .zip = archive, .exe = programme exécutable.
